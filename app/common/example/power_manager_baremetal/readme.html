Power Manager bare metal application

This example project demonstrates use of the Power Manager module in a bare metal application.

The application starts in awake mode (EM0). The user clicks on Button 1 to cycle through desired energy modes:

  • 1 click -> EM1
  • 2 clicks -> EM2
  • 3 clicks -> EM3
  • 4 clicks -> EM4

Button 0 is used to confirm selection.

Once a mode is selected, the device will remain in that mode until:

  • EM1, EM2: the sleep timer expires
  • EM3: a button is pushed
  • EM4: the device resets
